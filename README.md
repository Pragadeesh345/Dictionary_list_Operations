# Dictionary Based List Operations

This is a Python program where I used dictionary to perform different operations on a list.

The program calculates frequency of elements and performs some operations based on it using a menu driven system.

Functions in this program:
- Count frequency of elements
- Find duplicate elements
- Find highest frequency element
- Find lowest frequency element
- Find common elements between two lists

This program helped me understand how dictionary works and how it can be used in list problems.
